# Optimized Structures

This folder contains the optimized atomic coordinates of extended carbon nanocones (xCNCs).

## File Format

All structures are provided in XYZ format:

```
N
Comment line
Element1  x1  y1  z1
Element2  x2  y2  z2
...
```

## Contents

- **xCNC.zip**: Complete set of optimized nanocone structures

## Visualization

These files can be opened with:
- [VMD](https://www.ks.uiuc.edu/Research/vmd/)
- [VESTA](https://jp-minerals.org/vesta/)
- [Avogadro](https://avogadro.cc/)
- [Jmol](http://jmol.sourceforge.net/)

## Note

Place your `xCNC.zip` file containing the optimized `.xyz` structures in this directory.
