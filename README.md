This repository provides the optimized atomic structures of **extended carbon nanocones (xCNCs)** derived from non-graphene two-dimensional carbon allotropes. The structures were investigated using a combined **density functional theory (DFT)** and **reactive molecular dynamics** approach.

### What's Inside

We investigated four precursor monolayers: **acepentalene**, **naphthylene**, and the topologically distinct **r469-New20** and **r568** lattices. From these, fifteen nanocone structures with disclination angles of 60Â°, 120Â°, and 180Â° were constructed and fully optimized at the **GGA-PBE/DNP** level of theory.
